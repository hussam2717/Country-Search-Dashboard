# Country-Search-Dashboard
A simple and elegant web app to search and explore country information in real-time.

🔗 Live Demo - https://hussam2717.github.io/Country-Search-Dashboard/

🚀 Features
🔍 Search any country by name
🌎 Displays country details:
Country Code
Capital
Region
Population
Currency
Languages
⚡ Fast and responsive UI
🌙 Clean dark theme design

🛠️ Tech Stack
HTML5
CSS3
JavaScript
REST API (for country data)

📂 Project Structure
├── index.html
├── style.css
├── script.js

💡 How It Works
Enter a country name in the search box
Click on the Search button
View country details dynamically fetched from API
